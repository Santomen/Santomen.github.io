Hospital Interfaz

En este proyecto se diseñó una interfaz gráfica básica en Python para que usuarios no relacionados a la informática pudieran manejar información de una base de datos sin necesidad de conocer algún lenguaje de bases de datos, permitiendo: agregar,eliminar y modificar datos de la base.
En este caso se decidió implementar una interfaz para un hospital.

Proceso

Se buscaba resolver una problemática utilizando bibliotecas en Python utilizando POO, el equipo decidió facilitar las visualizaciones de datos con el uso de una interfaz, esta se desarrolló utilizando Tkinter, además para analizar datos de manera gráfica se integró Matplotlib.
Fue sin duda alguna un proyecto difícil de llevar a cabo ya que solo se dio 10 días para presentarlo y dentro de estos 10 días se llevó a cabo la planeación del proyecto, el cómo iba a funcionar y el aprender a usar las bibliotecas Tkinter y Matplotlib que para ese momento eran totalmente desconocidas para el equipo.
Para resolver la primera parte del problema no hubo mucha complejidad, desde un principio se desarrolló la idea sin la interfaz, después de esto se empezó a estudiar las librerías utilizadas, lo cual fue lo que conllevó más tiempo, una vez hecho esto se implementaron las bibliotecas al proyecto desarrollado, lo más difícil fue la implementación de la interfaz ya que cambió el diseño del software, ya que este paso a ser alrededor de un 90% de funciones.

Un poco sobre el funcinamiento de la interfaz

En la imagen superior derecha ingresa se muestra el menú "base de datos" en este se pide información de los 3 diferentes empleados del hospital que se permite ingresar datos, cada punto es validado por ello de los botones "validar".
En la imagen 1 está el menú donde se pueden visualizar los datos,en donde se busco el nombre de un médico y se eliminó, es por ello que no aparece en la gráfica de la imagen 3 y en la imagen 2 se ejecuta una opción del menú que permite visualizar distintos datos por medio de gráficas en este caso se comparan los suledos de todos los empleados registrados.
